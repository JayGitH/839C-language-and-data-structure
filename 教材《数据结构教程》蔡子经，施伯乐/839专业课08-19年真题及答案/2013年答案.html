题目

(1) create()用于创建文件，如果要创建的文件不存在，则使用参数prems指定的权限创建文件

(2) 若static修饰局部变量，则局部变量是静态局部变量，存储在静态存储区，只在定义时初始化一次，在语句体内保持上次运行的值，若修饰全局变量，则为静态全局变量，仅在定义时初始化一次，多程序结构中静态变量仅能在定义的程序中有效，对其它程序没有作用。若用于修饰函数，则为静态函数，它只能在当前的程序文件中使用，不能跨文件使用。

(3) 使用全局变量会降低函数的独立性和通用性，并降低程序的清晰性，不同位置的函数可能在对全局变量的调用时会导致程序出现计算混乱甚至错误。

(4) 第i行j列的元素为*(m+i*n+j)

(5) 这 与问题的具体要求有关。对于一个含N的元素的表要做M次查找，若使用二分搜索，复杂度为O(NlogN)+O(MlogN)，而用顺序搜索的复杂度为O(MN)
若M相对N较小，则在大O记号的运算下，二分搜索的复杂度是O(NlogN)，顺序搜索的复杂度为O(N)
如果M的规模与N相当，M不视为常数，O(M)=O(N)，那么二分搜索的复杂度为O(NlogN)+O(MlogN)=O(NlongN)，顺序搜索的复杂度为O(N2)，此时二分搜索更快

(3)

/*
* bsearch - binary search an array of elements
* @key: pointer to item being searched for
* @base: pointer to first element to search
* @num: number of elements
* @size: size of each element
* @cmp: pointer to comparison function
*/
void *search(void* key,void* base,size_t num,size_t width,
            int(*compare)(const void* elem1, const void* elem2))
{
    size_t start = 0, end = num;
    int result;

    while(start < end)
    {
        size_t mid = start + (end - start) / 2;
        result = compare(key, base + mid * width);
        if(result < 0)
            end = mid;
        else if(result > 0)
            start = mid + 1;
        else
            return (void *)base + mid * width;
    }
    return NULL;
}
